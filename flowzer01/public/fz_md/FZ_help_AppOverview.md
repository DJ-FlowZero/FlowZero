# FlowZero — Workflow Overview

> **FlowZero** is a context-calibration system. It reduces the cost of explaining yourself to AI, colleagues, and services — every session, every handover.

The app is a linear workflow guide. Each row below the header is a step, in order.

---

## The Chain

```
FlowSet → Gestalt → Prune → PUCK → Job Package → Prime AI
```

---

## Row 1 — FlowSet Definition
**Start here.** Create the entity — person, place, object, environment.
A FlowSet is three linked files: **Profile** (stable facts) + **Flexicon** (vocabulary) + **Sticky** (what's different now).

## Row 2 — Populate members
Fill in the three members of the FlowSet you just created.
- **Profile Editor** — label/value facts that rarely change
- **Flexicon Editor** — personal vocabulary and meaning handles
- **Sticky Editor** — situational modifiers, active flags, gaps

## Row 3 — Calibrate and review
- **Flow Calibration** — set the 18 BioLogic behavioral parameters (tone, challenge level, response style…)
- **Gestalt Viewer** — read-only compiled view at chosen visibility level (Public / Private / Secret)

## Row 4 — Build and deliver the PUCK
- **Create PUCK** — select rows from the Gestalt, optionally add Flow Calibration, export as `.txt`
- **PUCK Library** — confirm the PUCK landed; manage the registry
- **Compose Job Package** — combine a task brief + one or more PUCKs → single priming document for any AI

## Row 5 — Document and repeat
- **Text File Editor** — spec docs, glossary, code plan, daily blog
- **MD Editor** — help files, marketing content, presentations

---

## UI Mode

Controls the visibility filter applied across the app.

| Mode | Shows |
|------|-------|
| **Public** | Safe-to-share rows only |
| **Private** | Public + private rows |
| **Secret** | Everything — full context |
