# Gestalt Viewer

> The **Gestalt** is the compiled view of a FlowSet — all three members (Profile + Flexicon + Sticky) rendered together at a chosen visibility level.

Use this to verify exactly what a PUCK will draw from before you build one. What you see here is what gets exported.

---

## Visibility levels

| Level | What you see |
|-------|-------------|
| **Public** | Only public-flagged rows — safe to share openly |
| **Private** | Public + private rows — for trusted collaborators |
| **Secret** | All rows — full personal context, nothing hidden |

## This is read-only

The Gestalt is generated, not stored. Edit the source members (Profile / Flexicon / Sticky), then return here to review the result. The chain: **FlowSet → Gestalt → Prune → PUCK**.
