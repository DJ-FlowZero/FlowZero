# MD Editor

> Edit any `.md` file in `fz_md/` — marketing content, help docs, product pitch, presentations.

Raw/Preview toggle renders markdown inline. Same backup chrome and version history as Text File Editor.

---

## Workflow

1. Select a file from the dropdown
2. Toggle between **Raw** (edit) and **Preview** (rendered markdown)
3. **Save** — writes to server with timestamped backup
4. **↺ Reload** — discards changes, re-fetches from server

## Help content lives here

The `?` button docs you are reading now are `.md` files in `fz_md/`. Edit them here — no code changes needed. Content and UI are fully decoupled.
