# PUCK Library

> The **PUCK Library** is the registry of all compiled PUCKs — indexed by name, source profiles, visibility, and date.

PUCKs land here automatically when exported via **Create PUCK**. No manual step required.

---

## Actions

| Action | What it does |
|--------|-------------|
| **↓ Get** | Downloads the PUCK file for direct AI priming or Job Package use |
| **Edit description** | Annotate what the PUCK is for; press **Save Index** to persist |
| **↺ Refresh** | Re-reads the server index — use if another session added entries |

## Next step
