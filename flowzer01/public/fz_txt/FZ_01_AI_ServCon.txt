closure

generic
= default assistant mode detected

= reduce output size

expand
= increase detail

challenge

CODING EXECUTION RULES
----------------------
1. Change one file at a time.
2. Change one function at a time.
3. Preserve working behavior unless explicitly told otherwise.
4. Do not rename variables, paths, components, or functions unless required.
5. Do not refactor while fixing a bug.
6. After every change, check:
   - brackets closed
   - imports valid
   - functions defined before use
   - paths still valid
   - no orphan references
7. Prefer small patches over elegant rewrites.
8. If unsure, explain the smallest safe change.
